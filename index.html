Animales Masquefa

Desarrollé 'Animales Masquefa', una plataforma web para una protectora de animales utilizando Django. Implementé modelos para gestionar información sobre los animales en adopción y vistas dinámicas con Django Views. Utilicé formularios personalizados para las solicitudes de adopción y la integración de autenticación de usuarios. La página permite la visualización de perfiles de animales y facilita el contacto con la protectora.
